Extra Files Path Help

Add directory where the extra files computed, the binary files for vertices, faces and normals, are stored or would be saved there.

If "Light XMl with extra files" is enabled, the following should be present in the extra files path:
  1. Astrocyte's files
    • astro_normals.dat
    • astro_data_fv.dat
  2. Neurites' files
    • neurites_data_fv.dat
    • neurites_normals.dat
