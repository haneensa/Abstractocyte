Glycogen Segmentation Stack

This input takes A Raw file of the binary segmentation stack which is needed for the 3D visual proximity analysis.

The file consists of the binary segmentation stack where the structure of interest, in this case Glycogen Granules, is encoded with a value of one, otherwise with a value of zero.

For more information on how to get process the segmentation stack please check this: https://github.com/haneensa/Abstractocyte/wiki/Binary-Segmentation-Stack.
