Metadata (HVGX) Help

This input file includes metadata with hvgx extension of the input dataset which has contextual information regarding anatomical objects the data contains. An example of the metadata includes the function of each object (ex. inhibitory, excitatory), synapses and parent-child relationship information.

The following describe the required metadata format for Abstractocyte:

  1. axon metadata
    • ax, id, function (0:ex,1:in), is_mylenated, abs_node_id, name
  2. dendrite metadata
    • dn, id, function (0:ex,1:in), abs_node_id, name
  3. spine metadata
    • sp, id, psd_area, volume, dendrite_id, does_form_synapse, with_apparatus, has_glia_nearby, spine name
  4. bouton metadata
    • bo, id, vesicleNo, volume, surfaceArea, axon_id, name, is_terminal_branch, is_mitochondrion
  5. synapse metadata
    • sy, id, vol_node_id, abs_edge_id, axon_id, dendrite_id, spine_id, bouton_id, name
  6. glia metadata
    • gl, id, name
  7. segmentation metadata
    • sg, vast_id, flags, r,g,b, pattern1, r2, g2, b2, pattern2, anchorx, anchory, anchorz, parent_id, child_id, previous_sibiling_id, next_sibiling_id, collapsed, bboxx1, bboxy1, bboxz1, bboxx2, bboxy2, bboxz2, voxels, type, object_id, name
  8. mitochondria metadata
    • mt, id, node_id, PARENT_TYPE, parent_id, name
